This is a webpage for git recap!

I can remember how to:

  • create a folder in command line
  • create files in command line
  • create a git repository
  • add files to staging area
  • commit changes and add a message
  • review git logs
  • repositories can be replicated. They are linked and are referred to as remotes
  • Github is where you store your "origin" i.e. first remote
  • push repository to github. pull to replicate to local
  • git remote will display remotes
  • github will provide the command to create an origin
  • git clone then link to repository to clone folder to local.
